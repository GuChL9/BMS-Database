# 普通化学

## 课程介绍

本课程主要讲授化学的基本原理和概念，将浅薄涉及热化学，电化学，配位化学等等内容。

### 预修要求

**必须掌握：**

- 高中化学基础

**推荐拓展：**

- 无需拓展
