# 细胞与生物分子Ⅱ

## 课程概况

Ⅱ重点在于生物分子，由三个老师分别讲三个部分（蛋白质与核酸；代谢反应；遗传）。多做笔记，及时弄懂，每个部分都是比较硬的。
这门课程实际上是外院的生物化学等课程的合体
