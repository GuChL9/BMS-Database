# 物理化学

## 课程概况

|||
| :---: | :---: |
| 学分 | 4 |
| 周学时 | 2+2 |
| 预修 | 高等数学 |

在基础医学的培养方案中，物理化学的建议修读学期是大一春夏，这是因为提前修读物理化学对修读在大二秋冬的细胞与生物分子Ⅱ有一定帮助。但实际上把物理化学放在大二秋冬修读更合适。
